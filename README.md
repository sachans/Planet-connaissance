# Planet-connaissance

## About
A ruby program which gives information about planet and make it's picture in terminal

## Dependencies
```
Ruby >= 1.8.7
Rmagick or MiniMagick
```

## Installation
### Step 1: 
Go in sudo mode.
```
su 
```
### Step 2:
Install these packages
```
gem install csv awesome_print launchy
```
### Step 3:
Install Catpix
```
gem install catpix
```
### Step 4:
Run planets.rb file
```
ruby planets.rb
```

## Example

### Input
```
ruby planets.rb
```
